# Monte Carlo 千葉県面積推定

## 1. プロジェクトの目的
このプロジェクトは、国土数値情報の行政区域データ (GeoJSON) を用いて、モンテカルロ法により千葉県の面積を推定することを目的としています。
ランダムに生成した点がポリゴン（千葉県の境界）の内側に含まれる割合から、近似的な面積を算出します。

## 2. 環境構築方法
Python 3.x 環境が必要です。依存ライブラリは `requirements.txt` に記載されています。
以下のコマンドで必要なパッケージ（`shapely`, `pyproj` など）をインストールしてください。

```bash
pip install -r requirements.txt
```

## 3. スクリプトの実行方法
GeoJSONファイルを指定してスクリプトを実行します。

```bash
python monte_carlo_area.py N03-23_12_230101.geojson -n 200000
```

## 4. 引数の説明
`monte_carlo_area.py` は以下のコマンドライン引数を受け付けます。

- `geojson` (必須): 処理対象のGeoJSONファイルのパス。
- `-n`, `--num-samples` (任意): モンテカルロ法で生成する乱数のサンプル数。デフォルトは `200000` です。サンプル数が多いほど精度が向上しますが、計算時間は長くなります。
- `--seed` (任意): 乱数生成のシード値。デフォルトは `0` です。同じシード値を指定することで、再現性のある結果を得ることができます。

**ヘルプの表示:**
```bash
python monte_carlo_area.py --help
```

## 5. 現在の実装状況と今後の課題
**現在の実装状況:**
- GeoJSONからのポリゴン読み込み (`shapely`)
- 正積投影 (Albers Equal-Area) への座標変換 (`pyproj`)
- モンテカルロ法による面積推定処理の実装と、Shapelyの組み込みメソッドによる解析的面積との比較

**今後の課題:**
- **パフォーマンス改善**: 現在はPython標準の `random` モジュールと `shapely` の `covers()` メソッドを使用して逐次処理しているため、サンプル数が多い場合に時間がかかります。ベクトル化や並列処理（例: `numpy`, `geopandas` の利用）の導入による高速化が課題です。
- **フィルタリング機能**: 現在はGeoJSONに含まれるすべてのフィーチャ（ポリゴン）を結合して処理しています。特定の市区町村のみを対象とするようなフィルタリング機能の追加が検討されます。
